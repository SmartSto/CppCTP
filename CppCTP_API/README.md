CppCTP_API  
=========

## 开发环境
    CentOS 6.5, gcc:v4.8.0, cmake:v2.8.12.2

# v0.1  
* 完成了CTP API的C++基础代码编写，未做异常错误处理  
* 实现的功能有如下  
|==============================|  
|请输入您的操作编号  
|【u】:账户信息查询  
|【t】:账户资金查询  
|【s】:查询合约信息  
|【d】:查询交易信息  
|【h】:查询持仓信息   
|【i】:报单录入  
|【o】:报单查询  
|【a】:撤单操作  
|【e】:查询交易所代码  
|【b】:退出程序  
|==============================|
